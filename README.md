# Social-Media-Addiction-Analysis

# 📱 Student Social Media Addiction: A Dual-Tool Analysis Case Study

### 🎯 Project Overview
This project investigates the impact of social media usage on 1,000 students globally. By building this analysis in both **Power BI** and **Tableau**, I demonstrate technical versatility and the ability to maintain data consistency across different software environments.

---

## 🛠️ Software Proficiency & Technical Stack
- **Power BI:** Focused on **Star Schema Modeling**, advanced **DAX measures**, and **Bookmark/Selection Pane** interactivity for storytelling.
- **Tableau:** Focused on **LOD (Level of Detail) Expressions**, **Dashboard Composition**, and **Interactive Actions**.
- 🔗 **[View Interactive Tableau Dashboard](https://public.tableau.com/app/profile/gopal.sarkar/viz/SocialMediaAddictionDashboard_17734785612930/Dashboard1)**

---

Analyzing the inverse relationship between daily screen time and psychological wellness scores.

---

## 💡 Final Strategic Recommendations
1. **Digital Boundaries:** Schools should implement a **"5-Hour Warning"** policy to help students protect their academic focus.
2. **Age-Targeted Outreach:** Prioritize wellness workshops for the **18-22 age segment**, where the addiction-to-sleep-loss ratio is highest.
3. **Data-Driven Counseling:** Provide wellness teams with these dashboards to identify high-risk individuals before performance drops.
